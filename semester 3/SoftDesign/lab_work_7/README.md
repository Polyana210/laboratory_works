# Генерация и обработка исключений. Обработка исключений с использованием менеджеров контекста
### Ход работы
1. Напишите на языке программирования Python программу, которая
* содержит функцию f, которая принимает один параемтр x и возвращает вычисленное значение 1/x2−1−−−−−√ (если значение не определено, то функция должна генерировать исключение);
* в цикле 100 раз считывает введенную пользователем строку, преобразует её в вещественное число и вывод результат вычисления f от этого числа.
2. Добавьте в программу из пункта 1 обработку исключений:
* обработка исключений должна проходить внутри цикла, в котором производится считывание данных;
* должны отдельно обрабатываться исключения ValueError, OverflowError, ZeroDivisionError – во всех случаях программа должна выводить соответствующее сообщение об ошибке и продолжать работу со следующей итерации цикла.
3. Добавьте в программу полученную в пункте 2 обработку исключения KeyboardInterrupt – при обработке этого исключения программа должна выводить количество оставшихся итераций цикла и переходить к следующей. (Это исключение возникает при попытке пользователя прервать работу интерпретатора нажатием специльной комбинации клавиш – обычно, Ctrl+C)
4. Модифицируйте программу из пункта 3 так, чтобы при возникновении исключения ValueError внутри функции f выводилось сообщение отличное от того которое будет выводиться при возникновении исключения ValueError в другом месте.
5. Напишите новую программу на Python, которая
* использует менеджеры контектса и конструкцию with;
* завписывает в файл data.txt данные о количестве натуральных делителей первого 1000000 натуральных чисел;
* сведения о количестве натуральных делителей для каждого натурального числа записываются в отдельную строку;
* в строке сначал записывается число, а потом отделенное двоеточие количество его делителей.
6. Ответьте на контрольные вопросы
### Контрольные вопросы
1. Какие встроенные исключения не наследуют от Exception? В каких случаях они возникают?
2. Можно ли перехватить и обработать исключение SyntaxError, возникшее  из-за синтаксической ошибки в импортируемом модуле?
3. Каким образом можно указать для одного блока expect, что он перехватывает исключения нескольких типов?
4. В каких случаях в программе пункта 1 могут возникнуть исключения ValueError, OverflowError и ZeroDivisionError? Какие из них могут возникнуть в функции f, а какие при преобразовании введенной строки в число?
5. Как определить место возникновения исключения?
6. Что делает менеджер контекста возвращаемый функцией open при выходе из конструкции with?
